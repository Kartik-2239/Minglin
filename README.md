### Installation

Minglin is a simple terminal chat app for using ollama models

Demo Video
[![Watch the demo]](assets/demo.mp4)

```bash
cd Minglin
npm install
./node_modules/.bin/tsc -p tsconfig.json
node dist/src/main.js
```

Notes:
- Requires Node.js 18+.
- Ollama must be running on the background.
